# Progetto chatbot

Attivazione ambiente virtuale conda:

```bash
conda activate ChatBot
```

Se il modello non è trainato:

```bash
rasa train
```

Runnare le actions:

```bash
rasa run actions
```
Per parlare con il chatbot (porta 5006):

```bash
rasa shell --port 5006
```
